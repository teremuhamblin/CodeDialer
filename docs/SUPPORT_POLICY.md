🆘 Support Policy — CodeDialer

Cette politique définit comment le support est géré pour le projet CodeDialer.

---

📚 Documentation

La documentation officielle se trouve dans :

`
Docs/
`

---

🐞 Bugs

Les bugs doivent être signalés via :

`
Issues → New Issue → Bug Report
`

Voir : .github/ISSUETEMPLATE/bugreport.md

---

🌟 Demandes de fonctionnalités

Les demandes doivent être soumises via :

`
Issues → New Issue → Feature Request
`

Voir : .github/ISSUETEMPLATE/featurerequest.md

---

🔐 Problèmes de sécurité

Ne créez jamais d’issue publique.

Contactez :

`
security@codedialer.dev
`

Voir : .github/SECURITY.md

---

💬 Questions générales

Les questions peuvent être posées via une issue avec le label :

`
question
`

---

🧭 Versions supportées

| Version | Support |
|--------|---------|
| 1.x.x  | ✔️ Active |
| 0.x.x  | ❌ Non supportée |

---

❤️ Merci

Merci d’utiliser CodeDialer et de respecter cette politique de support.
`
